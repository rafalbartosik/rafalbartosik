**Project Overview:**  
This project analyzes e-commerce data from the Olist dataset (Kaggle) using Power BI. The goal was to extract actionable business insights regarding sales performance, logistics efficiency, and customer behavior.

## 🚦 Project: eZagrożenie – Road Risk Platform

🔗 **Live App:** https://ezagrozenie.streamlit.app  

**Project Description:**  
eZagrożeNIE is a web application built with Python and Streamlit designed to analyze road accident risk along specific routes within the Lubuskie Voivodeship. The system combines route optimization with historical accident data to assess safety levels and support risk-aware navigation decisions.

Users can generate routes between two points and evaluate the level of danger based on historical accident occurrences within a defined buffer zone.

---

**Key Features:**
- Route planning using OSRM API (primary and alternative routes)
- Accident analysis within a safety buffer along the route
- Interactive geospatial visualization using Folium
- Risk prediction model (Logistic Regression – scikit-learn)
  - based on time of day
  - day of week
  - month
- Detailed data exploration (location, street, distance from route)
- Toggleable data tables for deeper analysis

---

**Tech Stack:**
- Python 3.x
- Streamlit (web application framework)
- Streamlit Cloud (deployment)
- OSRM API (routing engine)
- Pandas, NumPy (data processing)
- Folium, Streamlit-Folium (map visualization)
- Scikit-learn (Logistic Regression model)

---

**Data Source:**
Historical road accident dataset (CSV format) covering Lubuskie Voivodeship (2018–2024)

---

**Technical Note:**
Using the full dataset (2018–2024) may increase computation time due to larger spatial processing load.

## 🚦 Projekt: eZagrożenie – platforma ryzyka drogowego

🔗 **Aplikacja:** https://ezagrozenie.streamlit.app  

**Opis projektu:**  
eZagrożeNIE to aplikacja webowa w Pythonie (Streamlit) analizująca ryzyko wypadków drogowych na trasach w województwie lubuskim. System łączy planowanie tras z analizą historycznych danych o wypadkach.

---

**Funkcje:**
- Wyznaczanie tras (OSRM API)
- Analiza wypadków w buforze bezpieczeństwa
- Wizualizacja map (Folium)
- Model predykcyjny (regresja logistyczna)
- Analiza danych lokalizacyjnych (miasto, ulica, dystans od trasy)

---

**Technologie:**
- Python 3.x
- Streamlit
- Pandas, NumPy
- Folium
- Scikit-learn
